# Дії над многочленами

* <p><b>Додавання/віднімання</b>. Потрібно рокрити дужки та звести подібні доданки. Якщо перед дужками стоїть знак $$"$$+$$"$$, то знаки дужок лишаються незмінними; якщо  $$"$$-$$"$$, то знаки всередині дужок слід замінити на протилежні.</p>
<p><i>Наприклад:</i> $$(4x^2 y + xy) + (x - 6 - xy) = 4x^2 y + xy + x - 6 - xy = 4x^2 y + x - 6.$$<br>$$(4x^2 y + xy) - (x - 6 - xy) = 4x^2 y + xy - x + 6 + xy = 4x^2 y + 2xy + x - 6.$$</p>
* <p><b>Множення одночлена на многочлен</b>. Потрiбно одночлен помножити на кожний доданок многочлена i додати отриманi добутки.</p>
<p><i>Наприклад:</i> $$(4x^2 y - xy)\cdot(-3y^3)=4x^2 y\cdot(-3y^3) + (-xy)\cdot(-3y^3 ) = -12x^2 y^4 + 3xy^4.$$</p>
* <p><br>Множення многочлена на многочлен</br>. Потрiбно кожен доданок одного многочлена помножити на кожен доданок другого многочлена i додати отриманi добутки.</p>
<p><i>Наприклад:</i> $$(4x^2 y-xy)\cdot(x-6-xy)=4x^2 y\cdot x+(-xy)\cdot x+4x^2 y\cdot(-6)+(-xy)\cdot$$<br>$$\cdot(-6)+4x^2 y\cdot(-xy)+(-xy)\cdot(-xy)=4x^3 y-x^2 y-24x^2 y+6xy-4x^3 y^2+x^2 y^2=$$<br>$$=-4x^3 y^2+4x^3 y+x^2 y^2-25x^2 y+6xy.$$</p>
* <p><b>Ділення многочлена на одночлен.</b> Потрібно кожен доданок многочлена поділити на одночлен і додати отримані частки.</p>
<p><i>Наприклад:</i> $(4x^5y-2x^3y^3+8xy^2+6xy):(2xy) = (4x^5y):(2xy) - (2x^3y^3):(2xy)+$$<br>$$+ (8xy^2):(2xy) + (6xy):(2xy) = 2x^4 - x^2y^2 + 4y + 3.$</p>
